Premier Ordre ​

Circuits Passifs ​

Passe-bas RC ​

La fonction de transfert pour ce filtre est donnée par :

HLP(s)=Vout(s)Vin(s)=11+sRC

Passe-haut RC ​

La fonction de transfert pour ce filtre est donnée par :

HHP(s)=Vout(s)Vin(s)=sRC1+sRC

Circuits Actifs ​

Passe-bas SP ​

La fonction de transfert pour ce filtre est donnée par :

HLP(s)=Vout(s)Vin(s)=−R2R11+R2Cs

Passe-haut SP ​

La fonction de transfert pour ce filtre est donnée par :

HHP(s)=Vout(s)Vin(s)=−R2Cs1+R1Cs

Second Ordre ​

Passe-bas RC/RC ​

La fonction de transfert pour ce filtre est donnée par :

HLP(s)=Vout(s)Vin(s)=11+(C2R2+C2R1+C1R1)s+R1R2C1C2s2

Passe-bande RC/RC ​

La fonction de transfert pour ce filtre est donnée par :

HLP(s)=Vout(s)Vin(s)=R2C1s1+(C2R2+C1R2+C1R1)s+R1R2C1C2s2

Passe-haut RC/RC ​

La fonction de transfert pour ce filtre est donnée par :

HHP(s)=Vout(s)Vin(s)=R1R2C1C2s21+(C2R2+C2R1+C1R1)s+R1R2C1C2s2

Passe-bas RLC ​

La fonction de transfert pour ce filtre est donnée par :

HLP(s)=Vout(s)Vin(s)=11+RCs+LCs2

Passe-bande RLC (forme 1) ​

La fonction de transfert pour ce filtre est donnée par :

HBP(s)=Vout(s)Vin(s)=RCs1+RCs+LCs2

Passe-bande RLC (forme 2) ​

La fonction de transfert pour ce filtre est donnée par :

HBP(s)=Vout(s)Vin(s)=LR1s1+L(R1+R2R1R2)s+LCs2

Passe-haut RLC ​

La fonction de transfert pour ce filtre est donnée par :

HHP(s)=Vout(s)Vin(s)=LCs21+RCs+LCs2
